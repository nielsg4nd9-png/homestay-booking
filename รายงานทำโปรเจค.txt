# หน้าหลัก (บ้านสวนโฮมสเตย์)
  - มีส่วนพรีวิว ห้องพัก

----------------------------------------------------------------------------------------------------------------

# จัดการห้องพัก
  - เพิ่ม/ลบ/แก้ไขข้อมูลห้องพัก
  - มีการจองผ่านหน้าห้องพัก

# การจัดการสมาชิกและสิทธิ์การเข้าถึง

- ล็อกอินด้วย admin@example.com / admin1234 (หรือค่าที่คุณตั้งอง)
- ADMIN สามารถเปลี่ยน role เป็น ADMIN / EMPLOYEE / USER
